可以简单的上手搭建本地开发环境。

## Step 1: 安装 Docker 和 Docker compose

安装参考文档 [docs.docker.com]。

## Step 2: 启动 Dragonfly

进入 Dragonfly 项目中并启动 docker-compose。

```bash
$ docker-compose up
Creating network "dragonfly2_default" with the default driver
Creating cdn ... done
Creating scheduler ... done
Creating dfdaemon  ... done
Attaching to cdn, scheduler, dfdaemon
```

## Step 4: 停止 Dragonfly

```bash
$ docker-compose down
Removing dfdaemon  ... done
Removing scheduler ... done
Removing cdn       ... done
Removing network dragonfly2_default
```
